📥 Step 1 — Download the App File
Everything is in one .html file. Download it to your laptop first.

1. In Claude, right-click the file link → Save Link As…
2. Save as pb-field-tawang.html to your Desktop.
🐙 Step 2 — Put it on GitHub Pages
GitHub Pages gives you a permanent HTTPS URL — required for the microphone to work in Chrome on Android.

A. Create account (skip if you have one)
Go to github.com → Sign Up → verify email.

B. Create a repository
Click ＋ (top-right) → New repository
Name: pb-tawang · Set to Public · Tick Add README → Create.

C. Upload the file
In the repo → Add file → Upload files → drag in pb-field-tawang.html → Commit changes.

D. Enable Pages
Settings tab → Pages (left sidebar) → Branch: main, folder: / (root) → Save.
Wait ~60 seconds. A green banner appears with your URL:
https://YOUR-USERNAME.github.io/pb-tawang/pb-field-tawang.html
📱 Step 3 — Open on Your Phone
Android → Chrome
Open Chrome (not Samsung Browser, not Firefox).
Paste the GitHub Pages URL → tap Allow when mic is requested.
To install: ⋮ menu → Add to Home Screen.

⚠️ Never open via Google Drive or as a local file — mic is blocked that way on Android. Always use the GitHub URL in Chrome.
iPhone → Safari
Open Safari (not Chrome on iOS).
Paste URL → Share → Add to Home Screen.
Tap Record → Allow mic when Safari asks.
💡 iOS uses non-continuous recognition — the app auto-restarts it silently. Just speak at a steady pace.
🔄 Step 4 — Updating Later
When you get a new version from Claude:

1. Go to your repo on github.com
2. Drag the new .html file onto the repo page → it asks to replace the old one.
3. Commit changes → live in ~60 seconds.

💡 Your field data lives in the phone's browser storage, not in the file — updating never deletes your records.
🎙 Recording Tips
· Open via the GitHub Pages URL in Chrome (Android) or Safari (iPhone)
· Hold phone 15–25 cm from mouth
· Speak at a steady, clear pace
· Short pause between sentences helps the engine finalise words
· If signal is weak, dictate in short bursts — the text is saved as each sentence finalises
· After recording, always scroll up and check the text before saving — Tawang place names (Zemithang, Shonga-tser, Gorichen) may need a quick fix
· Expected accuracy: 85–95% clear speech / good signal; drops in wind — find shelter or type manually
